# Projekt_Ausbildung_Falldetektor
Ablage der Projektunterlagen des Ausbildungsprojekts von Bosch Sicherheitssysteme
Erstellt im Dezember, Arbeitszeit ungefähr 8Std. 
Besteht aus Planungsunterlagen, 3D Model, Website, GitHub Repository

3D Model und Technische Zeichnungen können leicht abweichen von den Bemassungen. Das 3D Model ist so nicht druckbar und dient nur zum Anschauen und Exportieren. 

Planungsunterlagen Erfassen:
1. Zeichnungen
2. 3D Model
3. Code
4. Verbindungsplan
5. KiCat Datei des Projektes
6. Materialliste
7. 3D Druck
